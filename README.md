# TermSnap 📸 💻

**TermSnap** 是一款专为 macOS 设计的轻量级工具，集成了 **快速截图标注** 和 **Finder 快捷打开终端** 两大核心功能。它常驻于菜单栏，旨在提升开发者的日常工作效率。

---

## 🚀 核心功能

### 1. 智能截图与标注
- **窗口识别**：自动识别鼠标下的窗口边缘，支持高亮选区。
- **全屏快捷选中**：鼠标移动到空白桌面即可一键选中全屏。
- **Retina 适配**：完美支持 Retina 屏幕，截图保持 1:1 视觉比例且高清。
- **丰富标注**：支持矩形、箭头、文字、马赛克（像素化）等标注工具。
- **快捷操作**：支持 `回车` 复制到剪贴板，`Esc` 退出。

### 2. Finder 扩展 (Open Terminal Here)
- 在 Finder 中点击右键，直接在当前目录打开终端。
- 极速响应，无需手动输入 `cd` 命令。

## 🛠 编译与打包

本项目使用 Swift 5 编写，要求 macOS 14.6 或更高版本。

### 使用脚本构建 (推荐)
根目录下提供了一个自动化脚本，可以一键清理、构建并注册扩展：

```bash
chmod +x build.sh
./build.sh
```
打包后的 App 将存放在 `./build/Release/TermSnap.app`。

### 使用 Xcode 构建
1. 打开 `TermSnap.xcodeproj`。
2. 选择 `TermSnap` Scheme。
3. 点击菜单栏 **Product > Archive** 进行打包，或直接运行。

---

## 📦 安装与配置 (重要)

由于 macOS 的安全机制，Finder 扩展需要手动激活才能在右键菜单显示：

1. **移动应用**：将 `TermSnap.app` 拖入你的 **`/Applications` (应用程序)** 文件夹。
2. **首次运行**：双击打开一次 TermSnap。
3. **启用扩展**：
   - 打开 **系统设置 > 隐私与安全性 > 扩展 > Finder 扩展**。
   - 勾选 **TermSnap** 选项。
4. **重启 Finder**：如果右键菜单没出现，请在终端运行 `killall Finder`。

## ⌨️ 快捷键

- **截图模式**：
  - `Enter`: 完成并复制到剪贴板。
  - `Esc`: 取消截图。
